Research Objective

My research advances the computational understanding of urban mobility through innovative Data Science and Machine Learning methodologies that decode complex human movement patterns and their emergent collective behaviors. Operating at the intersection of Data Analytics, Urban Mobility, and Behavioral Economics, I leverage cutting-edge ML algorithms and statistical modeling techniques to analyze large-scale mobility datasets, investigating how individual transportation choices aggregate into system-wide patterns and how anomalies emerge within these dynamic networks.

My work addresses pressing societal challenges including urban congestion, public safety, and equitable access to transportation, while advancing our theoretical understanding of collective human behavior in spatiotemporal contexts. By leveraging advanced techniques such as Knowledge Graph Construction, Multi-Dimensional Ensemble Learning, and Transportation Mode Classification, I develop frameworks that can identify subtle patterns in mobility data, classify anomalous behaviors, and predict transportation choices under various conditions.

Real-Time Anomaly Detection

Developing interactive dashboards that detect behavioral anomalies in streaming mobility data using ensemble methods, achieving 93% accuracy in transportation mode classification and identifying 300 anomalous agents from large-scale simulated populations.

Knowledge Graph Intelligence

Creating semantic graph representations that capture temporal patterns, spatial relationships, and causal inference mechanisms. These graphs model agent interactions across 15+ different approaches, revealing emergent collective behaviors invisible when examining individual trajectories.

Crowd Pattern Analytics

Analyzing visitor flow patterns and rare event detection in urban environments, including the analysis of Friday prayer gatherings that show 5-10x normal visitor spikes at religious locations, contributing to Point-of-Interest analytics.

Behavioral Economics Applications

Applying Mechanism Design for Social Good and behavioral pattern recognition to understand how economic incentives influence mobility choices, particularly in scenarios involving deliberate obfuscation or "hiding" behaviors in transportation networks.

Key Technical Contributions

  • Multi-Dimensional Ensemble Framework: Developed novel approaches addressing both individual and group-level anomalies across distinct taxonomic categories (point, contextual, collective, and distributional anomalies)
  • Advanced ML Pipeline: Built comprehensive classification systems using Random Forest + Gradient Boosting ensembles, Gaussian Mixture Models, and DBSCAN outlier detection
  • Temporal Analysis Methods: Implemented rolling window analysis with 96-period (24-hour) windows for detecting visitor pattern anomalies and Z-score based rare event detection
  • Graph-Based Intelligence: Created 15+ knowledge graph approaches incorporating agent behaviors, temporal nodes, and edge weight variability analysis for distinguishing anomalous vs. non-anomalous patterns
